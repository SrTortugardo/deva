## FEATURES
### Dispositivos soportados
- ATA
- Keyboard

### Other things
- ELF
- FAT32
- Framebuffer
- GDT
- Paging
- Multitasking
- Virtual File System (or kinda, its only a wraper)
- Support for programs
- A C Library (wip)

### Supported CPUs
- i386/i686 : i dont know the name, this is the most supported arch
